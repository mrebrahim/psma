# بصمة شباب مصر — الموقع الرسمي

موقع مؤسسة **بصمة شباب مصر** (PSMA) — مبادرة وطنية تابعة لوزارة الشباب والرياضة.

الموقع تعريفي بالمبادرة + منصة وظائف (الشركات تنشر، والشباب يتقدّمون).

## التقنيات

- **Next.js 16** (App Router) + **TypeScript** + **Tailwind CSS 4**
- **Supabase** (PostgreSQL + RLS) لقاعدة بيانات الشركات والوظائف والتقديمات
- **خط Almarai** + اتجاه RTL بالكامل
- **Vercel** للاستضافة

## التشغيل المحلي

```bash
cp .env.example .env.local   # ثم املأ القيم
npm install
npm run dev
```

افتح http://localhost:3000

### متغيرات البيئة

| المتغير | الوصف |
|---|---|
| `NEXT_PUBLIC_SUPABASE_URL` | رابط مشروع Supabase |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | المفتاح العام (publishable) |

## الصفحات

- `/` — الرئيسية (تعريفية + عداد للملتقى القادم + أرقام الأثر + البرامج + الفريق + الشركاء)
- `/about` — من نحن (الرؤية، الرسالة، الإطار القانوني، الهيكل)
- `/programs` — البرامج
- `/jobs` — قائمة الوظائف المتاحة
- `/jobs/[id]` — تفاصيل الوظيفة + نموذج التقديم
- `/companies/register` — تسجيل الشركة
- `/companies/[id]/jobs/new` — نشر وظيفة جديدة
- `/contact` — تواصل معنا

## قاعدة البيانات

ثلاث جداول رئيسية مع RLS:

- `companies` — الشركات الناشرة للوظائف
- `jobs` — الوظائف المنشورة
- `applications` — تقديمات الباحثين عن عمل

## النشر على Vercel

1. ادفع المشروع لـ GitHub
2. اربط الـ repo بـ Vercel
3. أضف متغيري البيئة في إعدادات Vercel
4. النشر تلقائي
